# MVP

Esta carpeta define el recorte minimo viable de **Vera** para arrancar desarrollo sin cargar todo el alcance completo.

En este MVP solo opera el **intermediario** y el sistema se limita a:

- consultar la informacion concentrada de vehiculos;
- generar reportes;
- guardar y consultar archivos oficiales del vehiculo.
