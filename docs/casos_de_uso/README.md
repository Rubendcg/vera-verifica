# Casos de Uso

Los casos de uso describen como un actor interactua con el sistema para lograr un objetivo de negocio.

En Vera sirven para traducir el modelo de base de datos y los requerimientos del negocio en funciones concretas que despues pueden convertirse en modulos, APIs, reglas y pantallas.
